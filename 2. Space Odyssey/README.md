# Space Odyssey

A 3D Unity game with the objective of flying the rocket from the start platform to the end platform while clearing the obstacle course in between.

# Controls

[Spacebar] : Thrust

[A, D] : Left, Right rotation

# Levels
1. Mercury
2. Venus
3. Earth
4. Mars
